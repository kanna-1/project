What's one change we made that had a huge impact on how easy it is to create content regularly - and that our audience actually thanks us for?

👉 We stopped publishing assets.
👉 We started building formats.

We didn’t want to run a content factory: PDFs, blogs, one-off campaigns that live once and die.

Instead, we built a media machine:

1️⃣ Designing recurring formats (podcasts, live events, newsletters) buyers expect and return to.
2️⃣ Embedding internal SMEs as the voices, not ghostwritten stand-ins.
3️⃣ Mapping distribution loops - so every asset fuels paid, organic, sales enablement, and SEO.

The shift is simple: stop asking “what should we make this month?” and start asking “what’s our next episode?”

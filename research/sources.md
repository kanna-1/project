## Important Note On LinkedIn Collection

The LinkedIn script does **not** find posts automatically.

The workflow for LinkedIn is:
1. manually find a public LinkedIn post URL
2. pass that exact URL into `scripts/fetch_linkedin_post.py`
3. save the parsed post into `research/linkedin-posts/`

That means the current LinkedIn corpus reflects the exact public URLs that were manually supplied and then fetched with the script.
